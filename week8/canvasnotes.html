Canvas, SVG, and Drag and Drop Notes

  • Canvas allows you to draw shapes and lines, arcs and text, gradients and patters, as well as manipulate pixels in images and videos
    • Add a canvas element to the page; the text located between the html canvas tags will only be shown if the browser does not support canvas
    • Make sure to give the canvas tags an ID so the canvas JavaScript can access the element from the DOM
    • You must set both a width and height attribute to your canvas element; DO NOT USE CSS TO ADD THE ATTRIBUTES
    • Set Up The Canvas JavaScript File
      • Save the JavaScript File
      • Grab the canvas element on the page; var canvas = docuement.getElementbyId("myCanvas");
      • Set up the canvas context; var context = canvas.getContext("2d");
      • Set a color string to style the stroke; context.strokeStyle = "red";
        • Can be changed to be a pattern, gradient, or image
      • Define a fill color; context.fillStyle = "blue";
      • Add other elements such as drawing a rectangle; context.fillREct(10, 10, 100, 100); context.strokeRect (10, 10, 100, 100);
      • Paths - create a blueprint for your lines, arcs, and shapes, but the paths are invisible until you give them a stroke
      • toDataURL - creates a way you can save your canvas drawings as either a PNG or JPEG
      • Canvas can be used to draw on image on your site as well as manipulate the image; manipulations are looked at per pixel so you can use a for loop to iterate through every pixel in the image
    • SVG - Scalable Vector Graphics- Vector Images preserve their quality even as you blow them up or shrink them down
    • Raphael is an open source JavaScript library that makes drawing and animating with SVG's much easier
    • Drag and Drop API - allows us to specify that certain elements are draggable and then specify what should happen when these draggable elements are dragged over or dropped onto other elements on the page; not supported by Android or iOS
      • Set the drggable attribute on any HTML element you'd like to be draggable
      • Add an event listener for the dragstart event on any draggable HTML element
      • Add an event listener for the dragover and drop events on any elements that you want to have accept dropped items
