Arrays, Logic, and Loops Notes

  • Arrays - Ordered lists of values
  • Manipulating Arrays
    • .pop() – removes the last item from an array
    • .shift() – removes the first item from an array
    • .push() – appends new value at the end of the array
    • .unshift() – appends new value at the start of the array
    • .join() – turns the array into a list of stings separated by commas ( the separator can be changed by placing it in parentheses .join(‘&’)
    • .slice() – creates a subarray by chopping out a slice of the original array; used to delete data from array as well as append in specific places
    • .reverse() – permanently reverses order of the array
    • .sort() – permanently arranges the array alphabetically
    • .indexOf() – find the first occurrence of a value in an array; if not in the array displays -1
    • .includes() – a Boolean version of indexOf()
  • Sets - Data structure that represents a collection of unique values; cannot have any duplicate values
  • Manipulating Sets
    • Creating Sets const list = new Set();
    • .add() - adds values to a set
    • .size() - determines the number of values in a set
    • .delete() - removes values from a set
    • .clear() - removes all values from a set
    • [...] - Spread Operator; converts the set into an array literal
  • Maps - Data structure that keeps a list of key and value pairs
  • Manipulating Maps
    • Creating Maps const romanNumerals = new Map();
    • .set(1, 'I') - add a key and value pair to a map
    • .get() - looks up a value in a map
    • .has() - check if a particular key is in a pay; returns a boolean value
    • .delete() - remove a key and value pair from a map
    • .clear() - remove all key and value pairs from a map
    • Array.from() or spread operator [...] can be used to convert a map into a nest array of key-value pairs
  • If Statement - The code inside the block will only run if the conditions in parentheses is true
    • if (condition) {
      code to run if true
      }
  • Else Statement -Add an alternative block of code to run if the condition is false
    • if (condition) {
      code to run if true
      } else {
      code to run if condition is false
      }
    • Ternary Operator - Shorthand way to write an else statement
      • condition ? (//code to run if condition is true) : (//code to run if condition is false)
  • Switch Operator - Used When your code has lots of conditions to test
    • switch (number) {
      case4:
      console.log('You rolled a four');
      break;
      case 5:
      console.log('You rolled a five');
      break;
      case 6:
      console.log('You rolled a six');
      break;
      default:
      console.log('You rolled a number less than four');
      break;
      }
  • Loops - Repeat a piece of code over and over again according to certain conditions; if the conditions are not met the loop will get stuck in an infinite loop until it crashes the program
  • While Loop - repeatedly runs a block of code while a certain condition is true
    • while (condition) {
      do something
      }
  • Do...While Loop - similar to a while loop but the conditions come after the block of code; loop always runs once regardless of the condition being true or not
    • do {
      do something
      } while (condition)
  • For Loop - has three parts initialization code that runs before the loop starts, a condition that has to be satisfied for the loop to continue, and after code that is what to date after each loop iteration
    • for (initialization ; condition ; after {
      do something }
    • For Loops can be nested inside another loop so that the inner loop will run all the way though before the next step of the outer loop occurs
    • For loops can be used to loop over arrays, sets and maps
