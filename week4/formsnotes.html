Forms Notes

  • document.forms = returns an HTML collection of all the forms in the document in the order they appear
    • const form = document.forms.search;
    • const form = document.forms['search'];
    • const [input, button] = form.elements;
    • const input = form['searchinput'];
  • form.submit() = will submit the form automatically; does not trigger submit events
  • form.rest() = reset all the form controls back to their initial values specified in the HTML
  • form.action()= set the action attribute of the form; can also be sent to a different URL to be processed on the server

  • Focus and Blur can be used to create place holder text on forms
  • Form Controls
    • <input> = fields, text, passwords, check boxes, radio buttons, and file uploads
    • <select> = menues of drop down categories
    • <textarea> = elements for longer text entry
    • <button> = elements for submitting or setting the form
  • Form Validation - The process of checking whether a user has entered the information into the form correctly; can occur on the client side using JavaScript or on the server side,; best to use JavaScript to improve the user experience when completing the form or to give feedback about errors before being submitted
